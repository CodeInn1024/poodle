# poodlejs
一个简单的模块化JavaScript 实用工具库

## 安装

### npm（推荐方式）
```bash
npm i poodlejs
```
## 使用

### vue 中使用
```js
import poodlejs from 'poodlejs';
Vue.prototype.$poodlejs = poodlejs;
```
